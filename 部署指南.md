## 🎯 10 分钟快速部署

### 第 1 步：运行部署脚本（2 分钟）

**双击运行：**
```
deploy-railway.bat
```

或手动执行：

```bash
cd C:\Users\Kyle\.homiclaw\workspace\mahjong-score-online

# 初始化 Git
git init

# 添加所有文件
git add .

# 提交
git commit -m "Deploy to Railway"
```

### 第 2 步：创建 GitHub 仓库（3 分钟）

1. 访问 https://github.com/new
2. 填写：
   - **Repository name**: `mahjong-score-online`
   - **Description**: 在线多人麻将计分系统
   - **Public** ✅（免费账号必须选 Public）
3. 点击 **"Create repository"**

### 第 3 步：推送代码（2 分钟）

复制 GitHub 显示的推送命令，通常是：

```bash
git remote add origin https://github.com/你的用户名/mahjong-score-online.git
git branch -M main
git push -u origin main
```

### 第 4 步：部署到 Railway（3 分钟）

1. 访问 https://railway.app
2. 点击 **"Sign Up"** 或 **"Login"**
3. 使用 GitHub 账号登录
4. 点击 **"Start a New Project"**
5. 选择 **"Deploy from GitHub repo"**
6. 选择 `mahjong-score-online` 仓库
7. 点击 **"Deploy Now"**

### 第 5 步：生成域名（1 分钟）

1. 部署完成后，点击项目卡片
2. 点击顶部 **"Settings"** 标签
3. 滚动到 **"Networking"** 部分
4. 点击 **"Generate Domain"**
5. 获得你的公网 URL，例如：
   ```
   https://mahjong-score-production.up.railway.app
   ```

## 🔧 常见问题

### Q1: Railway 登录不上？

**解决：**
- 使用 GitHub 账号登录（推荐）
- 检查网络，可能需要代理

### Q2: 部署失败？

**检查：**
1. Railway 面板查看 Build 日志
2. 确认 `package.json` 有 start 脚本
3. 确认代码已推送到 GitHub

### Q3: 访问域名显示错误？

**解决：**
1. Railway 面板查看 Deploy 日志
2. 确认服务状态是 "Running"
3. 等待 1-2 分钟让服务启动

### Q4: WebSocket 连接失败？

**检查：**
- 使用 HTTPS 访问（不是 HTTP）
- 清除浏览器缓存
- 检查浏览器控制台（F12）

### Q5: 免费额度是多少？

**Railway 免费计划：**
- 每月 $5 额度
- 足够个人项目使用
- 可在面板查看使用量

## 📞 技术支持

遇到问题？

1. **查看 Railway 日志**
   - 项目面板 → Deployments → View Logs

2. **检查代码**
   - 确保 server.js 使用 `process.env.PORT`
   - 确保 public 目录存在

3. **测试本地**
   ```bash
   npm install
   npm start
   # 访问 http://localhost:3003
   ```
