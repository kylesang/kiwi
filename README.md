# 🀄 麻将计分 - 在线多人版

支持多人实时同步的麻将计分系统！

## ✨ 核心特性

- 🌐 **多人在线** - 支持 4 人实时同步计分
- 🔄 **实时更新** - WebSocket 技术，毫秒级同步
- 📱 **跨设备** - 手机、平板、电脑都能用
- 🏠 **房间系统** - 创建/加入房间，私密对战
- 🔗 **分享邀请** - 一键生成链接邀请好友
- 💾 **自动保存** - 服务器存储，不掉线
- 🎨 **精美界面** - 现代化渐变设计

## 🚀 快速开始

### 方式 1：本地运行

```bash
# 进入项目目录
cd mahjong-score-online

# 安装依赖
npm install

# 启动服务器
npm start

# 访问 http://localhost:3003
```

### 方式 2：双击启动

```
双击 start.bat
```

### 方式 3：部署到云端

**部署到 Vercel/Railway：**

1. 上传代码到 GitHub
2. 在 Vercel/Railway 导入项目
3. 自动部署完成

## 📖 使用指南

### 创建/加入房间

**房主：**
1. 打开应用
2. 输入名字，选择座位
3. 房间号留空 → 点击"加入/创建"
4. 系统自动生成房间号
5. 点击"分享"邀请好友

**玩家：**
1. 打开房主分享的链接
2. 或输入房间号
3. 输入名字，选择座位
4. 点击"加入/创建"

### 计分操作

| 操作 | 说明 |
|------|------|
| **+1/-1/-5** | 快速加减分 |
| **🀄 胡牌** | 登记胡牌（自摸/点炮等） |
| **📋 记录** | 查看历史计分 |
| **👥 分享** | 生成链接邀请好友 |
| **🔄 重置** | 重新开始一局 |

### 胡牌类型

- **自摸** - 三家付分
- **点炮** - 点炮者付分
- **杠上花** - 自摸 x3
- **抢杠胡** - 自摸 x3
- **海底捞月** - 自摸 x3

## 🔧 配置说明

### 端口配置

默认端口：`3003`

修改方法：
```bash
# 方法 1：修改 server.js 中的 PORT
const PORT = 3003;

# 方法 2：使用环境变量
PORT=8080 npm start
```

### 房间过期时间

默认：24 小时

修改方法（server.js）：
```javascript
const expireTime = 24 * 60 * 60 * 1000; // 改为其他时间
```

## 🌐 部署指南

### 部署到 Railway

1. 创建 GitHub 仓库并上传代码
2. 访问 https://railway.app
3. 创建新项目 → Deploy from GitHub
4. 选择仓库 → Deploy
5. 获取公网 URL

### 部署到 Vercel

1. 安装 Vercel CLI：`npm i -g vercel`
2. 项目目录运行：`vercel`
3. 按提示完成部署

### 自有服务器

```bash
# 安装 Node.js
# 克隆代码
git clone <repo>

# 安装依赖
npm install

# 启动（后台运行）
pm2 start server.js --name mahjong-score

# 或直接用 nohup
nohup node server.js &
```
